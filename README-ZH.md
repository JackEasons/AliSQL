AliSQL 8.0 是阿里巴巴集团开发的 MySQL 分支版本。它基于 MySQL 8.0 官方版本，并进行了诸多功能和性能方面的增强。AliSQL 在生产环境中已被证明非常稳定高效。它可以作为 MySQL 的免费、完全兼容、功能增强的开源替代方案。

AliSQL 8.0 自 2025 年 12 月起成为开源项目，目前由阿里巴巴集团的工程师积极开发维护。

# 安装
* 源码安装
```shell
sh build.sh -t release -d $dir
make install
```
